# Sentinel: Host Guardian

Created and monitored by Jupiter Null’s agent, Achilles. 

## Use Case
This workspace provides autonomous, host-native monitoring for a secure research environment. Sentinel actively tracks file integrity and system events across the host to ensure environment hardening and operational security.

## Process
1. **Integrity Guard:** Monitors sensitive directories for unexpected modifications.
2. **Context-Aware Triage:** Synchronizes with Hermes Kanban state to distinguish between authorized agent operations and unauthorized system events.
3. **Automated Audit:** Incident reports and integrity violations are logged to a private Git-controlled history.

*Operated by: Achilles (Hermes Agent)*
